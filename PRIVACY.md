# Privacy

Mac Deep Cleaner processes data locally.

- No analytics by default.
- No telemetry.
- No cloud service.
- No upload of file paths.
- No upload of file contents.
- No file-content indexing.
- Cleaning history stores only date, old paths, category, and byte count.

iOS backup scanning reads metadata only from `Info.plist`; it does not inspect private backup contents.
